## SnippetsTeaching

This repository serves as a foundational starting resource for learning **FRC (FIRST Robotics Competition)** programming in java. It provides a curated collection of concise Java snippets designed to build a knowledge of basic syntax and robot-specific syntax implementations. 

> [!NOTE]
>This is not intended to be a sole resource but a helpful aid to introduce new topics before mastering.

Each module includes clear, documented code and explanations to help student programmers master essential concepts quickly. 

> [!NOTE]
> Leading letters are used to force intended file reading order.

### Key Learning Modules

*   **Basics:** Deep dives into `DataTypes.java`, `Variables.java`, and `zIfs.java` to establish a solid base understanding of typed langauges and logic flow.
*   **Intermediate:** Practical examples of `DataTypesAdvanced.java`, `Functions.java`, and `Switches.java` introducing ways to clean up and improve readability of code.
*   **Advanced:** Introduction to `DataTypesAgain.java`, `structs.java`, `tScoping.java`, and `uClasses.java` to start understanding OOP and subsystem architecture.
*   **Advanceder:** Implementation of `Interfaces.java` to enhance the basics of OOP for Advantage Kit based architecture.

---

**How to Use This Repo**
1.  Navigate to the relevant `.java` file for the concept you are studying.
2.  Review the commented code to understand the "why" behind the syntax.
